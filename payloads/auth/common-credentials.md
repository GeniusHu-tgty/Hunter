# Common Credentials - Quick Reference

## Default Credentials

### Web Applications
```
admin:admin
admin:password
admin:123456
admin:admin123
root:root
root:toor
test:test
guest:guest
user:user
```

### Databases
```
mysql:root:
postgres:postgres
oracle:oracle
mssql:sa:
redis:(no auth)
mongodb:(no auth)
elastic:(no auth)
```

### CMS
```
wordpress:admin:admin
drupal:admin:admin
joomla:admin:admin
```

### Network Devices
```
cisco:cisco
admin:password
admin:admin
root:root
```

## Common Password Patterns

### Username-based
```
admin123
admin2024
password123
letmein123
welcome1
```

### Keyboard Patterns
```
qwerty
qwerty123
1qaz2wsx
zxcvbnm
asdfghjkl
```

### Date-based
```
2024!
2024#
Jan2024
Summer2024!
```

### Company-based
```
companyname123
companyname!
companyname2024
```
